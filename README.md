# 2026-sousakuten-info
This repository contains the source code for the 2026 Sousakuten Information System, a web application designed to manage and disseminate information related to the 2026 Sousakuten event. The system includes features for handling announcements, deductions, and class information.

# 2026創作展情報共有
このリポジトリは、2026年の創作展に関連する情報を管理・共有するためのウェブアプリケーションのソースコードを含んでいます。システムは、伝達内容、減点情報、クラス情報などの機能を提供しています。
